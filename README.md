# 💪 Fitness App

Современное Android-приложение, разработанное с использованием **Kotlin** и **Jetpack Compose**, предназначенное для просмотра списка тренировок. Проект демонстрирует современные подходы к разработке Android-приложений, включая работу с REST API, архитектуру MVVM и реактивный интерфейс.

## ✨ Возможности

- 💪 Просмотр списка тренировок
- 🔎 Поиск тренировок по названию
- 🏷️ Фильтрация тренировок по типам
- 📋 Просмотр подробной информации о тренировках
- 🌐 Получение данных через REST API
- ⚡ Реактивный интерфейс на Jetpack Compose
- 🏗️ Архитектура MVVM
- 📦 Паттерн Repository
- 🔄 Асинхронная обработка данных с Kotlin Coroutines
- 📄 Сериализация JSON с использованием `kotlinx.serialization`
- 🚦 Обработка состояний загрузки, успешного получения данных и ошибок
- 🎬 Просмотр видео с использованием Media3 ExoPlayer

## 🛠️ Используемые технологии

| Категория | Технология |
|-----------|------------|
| Язык | Kotlin |
| UI | Jetpack Compose |
| Архитектура | MVVM |
| Сетевое взаимодействие | Retrofit |
| Сериализация | kotlinx.serialization |
| Асинхронность | Kotlin Coroutines |
| Управление состоянием | ViewModel |
| Видео | Media3 ExoPlayer |
